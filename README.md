# AI Startup Website

**AI Startup Website** is a modern, responsive, and professional landing page built for showcasing AI products, services, and solutions. It is fully built using **Bootstrap 5** and custom CSS, with smooth animations powered by **AOS (Animate On Scroll)**.

## ðŸ”¹ Features

* Fully responsive design, works on mobile, tablet, and desktop.
* Hero section with CTA buttons.
* Services section highlighting AI offerings.
* Features section with hover effects.
* How It Works section.
* Testimonials section for client feedback.
* Pricing section with popular plan highlight.
* FAQ and Final CTA section.
* Footer with quick links and contact information.
* Smooth scroll animations using AOS.
* Custom color theme, no default Bootstrap colors.
